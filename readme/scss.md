# SCSS

- CSS의 부족한 부분을 보완하기 위해 설계된 CSS 확장 언어
- Sass(Syntactically Awesome Style Sheets)의 구문 중 하나

## 주요 특징

- **네스티드 규칙**
  - 선택자를 중첩하여 작성 가능
  - 코드 구조를 명확히 하면서도 코드의 가독성을 높임
- **변수 재사용**
  - 선언한 변수를 재사용하여 일관성 유지
- **계산 기능**
  - CSS 불가능한 수치 계산을 지원하여 동적인 스타일을 작성 가능

## 사용법

1. SCSS 파일을 작성하려면 `.scss` 확장자를 사용합니다.
2. SCSS 컴파일러를 사용하여 SCSS 파일을 CSS 파일로 변환합니다.

자세한 내용은 [Sass 공식 문서](https://sass-lang.com/documentation)에서 확인할 수 있습니다.
